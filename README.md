
# MKL Grocery List (mk)

A command-line grocery list manager built in Python.

Create, update, search, export, and manage your grocery items easily from the terminal or future UI.  
This project supports both interactive CLI use and single-command automation.

---


## Features

- Add, remove, edit grocery items
- Set priority, cost, amount, and store
- Mark items for purchase
- Export a list of items to buy with total cost
- CLI mode and interactive prompt
- Ready for future GUI integration
